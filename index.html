Panel de Avisos Académicos

Librería de Componentes Visuales con JavaScript

Este panel utiliza la librería miniui.js para mostrar ventanas emergentes y notificaciones visuales reutilizables.

Avisos importantes

En esta sección se muestran avisos académicos mediante ventanas modales. El mismo componente puede reutilizarse con diferente título, mensaje y botón.

Notificaciones del sistema

Las notificaciones tipo toast sirven para informar al usuario sobre el resultado de una acción sin interrumpir completamente la navegación.
